# 🛒 ShopSphere

ShopSphere is a desktop-based e-commerce application built using Java Swing.  
It simulates a basic shopping experience where users can browse products, manage a cart, and perform a simple checkout.

---

## 🚀 Features

- 🧾 Add and manage products
- 🔍 Search and browse products
- 🛒 Add items to cart
- ❌ Remove items from cart
- 💳 Checkout with total price calculation

---

## 🛠 Tech Stack

- **Java**
- **Java Swing (GUI)**
- **File Handling (for data persistence)**

---

## 📂 Project Structure

src/
└── com.shopsphere
├── model # Product, Cart classes
├── ui # All UI screens (Home, Products, Cart)
├── util # File handling logic
└── Main.java # Entry point
