## 🚀 Features

- 🧾 Add and manage products  
- 🔍 Search and browse products  
- 🛒 Add items to cart  
- ❌ Remove items from cart  
- 💳 Checkout with total price calculation  

---

## 🛠 Tech Stack

- **Java**
- **Java Swing (GUI)**
- **File Handling (Serialization)**
